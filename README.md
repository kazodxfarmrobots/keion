# KEION LINK

軽音楽部の仲間を集めて学園祭ライブを目指す、ギャルゲー風のWebゲームです。  
`index.html` を起点に、GitHub Pages / Render へそのまま配置して動作します。

## 動作環境
- Google Chrome (PC / Android / iOS)
- HTML / CSS / Vanilla JavaScript のみ（外部JSライブラリ不使用）
- スマホ横向き（ランドスケープ）にも対応

## 実行方法
1. このリポジトリを静的ホスティングに配置
2. `index.html` を開く

## 操作方法
- タイトル画面: クリック/タップ または `↑` `↓` `Enter`
- 会話進行: クリック/タップ または `Enter` / `Space`
- ギター勧誘ミニゲーム（2Dスクロール追跡）:
  - `JUMP` ボタン / `Space` / 画面タップでジャンプ
  - 障害物を避けて盗難犯に追いつくとクリア
  - クリアタイムでランク決定（S:16.0秒以内 / A:28.0秒以内 / B:40.0秒以内）
  - 捕まえられない場合は最低ランク
- 学園祭ミニゲーム:
  - 赤い声（マイナス）はタップ/クリックして消す
  - 緑の声（プラス）は基本的に触らず流す
  - 声は上から下へ流れる
  - 下に到達した時に判定（緑: スコア+ / ムード少し+、赤: スコア- / ムード-）
  - タップ判定（赤タップ: 少し+、緑タップ: マイナス）
  - 20秒ごとにキャラのカットイン演出が発生
  - 開始前にミニゲーム内で3曲からBGMを選択（スターチェンジ / 黒い幕が上がる夜 / 命火のレクイエム）
  - 観客ムードゲージ
    - マイナスを消す（タップ）: ムード上昇
    - マイナスが下に届く: ムード低下
    - プラスが下に届く: ムード少し上昇
  - コンボルール
    - 成功判定が続くとコンボ上昇
    - コンボ中はムード上昇量にボーナス
    - 失敗判定でコンボは0にリセット
  - 制限時間は曲ごとに固定
    - スターチェンジ: 1分26秒
    - 黒い幕が上がる夜: 1分29秒
    - 命火のレクイエム: 1分08秒
  - スコアが `TARGET` 以上で演奏成功
- BGM: タイトルまたはゲーム画面の `BGM` ボタン
  - 通常時（演奏ミニゲーム以外）は `放課後の太陽光線` を再生

## ステータス
- `GUITAR` / `BASS` / `DRUM` の3項目を星（★☆☆〜★★★）で表示
- 仲間加入時にステータスを付与
  - `GUITAR`: 追跡ミニゲームのランク（S/A/B）に応じて付与
  - `BASS` / `DRUM`: 勧誘の成績（ミス回数）に応じて付与
- 演奏ミニゲームにも反映
  - 星が高いほど、赤声の出現率低下・声の落下速度低下・目標スコア緩和
  - 星が高いと加点時の獲得点が上がる

## 音声再生について
Chromeの自動再生制限に対応するため、BGMはユーザー操作（クリック/タップ/キー入力）後に再生開始します。
